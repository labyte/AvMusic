# 群晖开发文档

## 群晖Webapi

访问地址：https://domain.com:5001/webapi/{path}

path: 每个api对应的地址，通过 `SYNO.API.Info` 获取


每次请求除了地址和API，还需要其他载荷参数，常见的有：

- limit: 整型，一般为请求的数量
- offset: 请求的便宜量
- method: 可选值：list/getinfo/query 等
- library: 可选值：shared/aill/person
- api: API的名称，如：SYNO.AudioStation.MediaServer
- additional: 一般请求歌曲时，表示请求需要包含的信息，如：song_tag,song_audio，多个用 `,` 隔开
- version: 版本号，这个需要在要求的版本内
- sort_by: 根据那个字段进行排序，如: name
- sort_direction: 正反叙：ASC/DESC
- 其他


### SYNO.API.Info


获取当前DSM版本可用的API，通过此API确定其他API可用的版本参数和路径。

此API不需要登录信息，是一个开放的API，也是获取其他API的前提，

根据这个无法使用，还需要填写各种参数，这个需要自己在网上爬（大多数可以通过群晖网页端的播放器爬取）。


API信息：

```
    "SYNO.API.Info": {
      "maxVersion": 1,
      "minVersion": 1,
      "path": "entry.cgi",
      "requestFormat": "JSON"
    },
```


- 请求地址示例： https:192.168.0.123:5001/webapi/entry.cgi
- 请求方法：post
- 载荷
  - api: SYNO.API.Info
  - method: query
  - version: 1
  - query: all

- 请求连接(示例)：https://192.168.0.123:5001/webapi/query.cgi?api=SYNO.API.Info&version=1&method=query&query=all
- 响应： [响应JSON](../ApiDocuments/Response/00%20-%20Api.json)

#### （1）获取锁定(PIN)的项目


载荷

- api: SYNO.AudioStation.Pin
- version: 1
- method: list
- offset: 0" 
- limit: -1" 

- 响应： [响应JSON](/Response/02%20-%20主页pin.json)


#### （2）锁定(PIN)项目

可以锁定文件夹、专辑、播放列表、歌手等

- 载荷
    - api: SYNO.AudioStation.Pin
    - method: pin
    - version: 1
    - items:
   
    ```json
    [
        {
            "type":"album",
            "criteria": {
                    "album":"Ultrasound 乐之路 1997-2003",
                    "album_artist":"陶喆"
                },
            "name":"Ultrasound 乐之路 1997-2003"
        }
    ]
    ```
    这是锁定播放列表时items的值 `[{"type":"playlist","criteria":{"playlist":"playlist_personal_normal/热门"},"name":"热门"}]`
    

- 响应：

```json
{
    "data": {
        "errors": [
            0
        ],
        "items": [
            {
                "criteria": {
                    "album": "Ultrasound 乐之路 1997-2003",
                    "album_artist": "陶喆"
                },
                "id": "9",
                "name": "Ultrasound 乐之路 1997-2003",
                "type": "album"
            }
        ]
    },
    "success": true
}
```


#### （3）取消锁定(PIN)的项目

每个锁定的项目都有id，取消锁定时，将id加入请求集合

载荷

- api: SYNO.AudioStation.Pin
- method: unpin
- version: 1
- items: ["5"]


返回

```json
{"data":{"errors":[],"items":["5"]},"success":true}
```
